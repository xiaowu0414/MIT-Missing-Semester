[MIT-Missing-Semester 课程连接](https://missing.csail.mit.edu/2020/)

---

这个系列课程体感上目的是为了帮你形成一整套的工作流，或者一些实用技巧，以提高计算机相关工作的效率。

- Motivation: 
  - Introduces some commonly used tools and demonstrates their basic usage. 
  - Details will not be covered; only basic but frequently used functions will be explained.

如果之前没有接触过这些东西，可能第一次尝试使用会有些不习惯，但请相信，这真的会提高你未来的工作效率。
